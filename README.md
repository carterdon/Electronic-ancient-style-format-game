## 📖 项目简介
SBTI (Cyber Personality Type Indicator) 是一款专为 2026 年互联网生态打造的“人格确诊” H5 游戏。不同于传统的严肃心理测试，它结合了 **MBTI 的底层逻辑**与**当代赛博废话文学**，旨在通过 24 道灵魂拷问，测算你在网络海洋中的真实成分。

## ✨ 核心特性
- 🌑 **暗黑高级感 UI**：专为熬夜党设计的护眼配色，毛玻璃质感，丝滑动画。
- 🎯 **高精度 24 题库**：每个维度 6 道交叉验证题，涵盖 AI、对线、偷感、收藏等全场景。
- 📖 **万字深度解析**：
  - **赛博人设**：你在公海里的面具。
  - **社交偏好**：你的消息回复速度与风格。
  - **精神状态**：确诊你的“淡淡综合症”或“显眼包倾向”。
  - **生存指南**：给你的赛博生活开一份药方。
- 🔙 **人性化 UX**：支持回看上一题，拒绝手滑，从容确诊。
- 🧩 **16 型人格全覆盖**：从“云端扫地僧”到“节奏魔术师”，总有一款属于你。

## 🛠️ 技术实现
- **零依赖架构**：原生 HTML5 + CSS3 + Vanilla JS，单文件运行，无需 Node.js 环境。
- **响应式适配**：完美兼容各型号移动端设备与 PC 浏览器。
- **状态管理**：内置历史栈逻辑，实现非线性测试体验。

## 🚀 快速部署
由于是纯静态页面，你可以通过以下方式秒速开启：

1. **直接运行**：下载 `index.html`，双击浏览器打开。
2. **GitHub Pages**：
   - Fork 本仓库。
   - 在 Settings -> Pages 中开启服务。
3. **Vercel / Netlify**：直接将文件夹拖入即可获得永久 HTTPS 链接。

## 🧪 测算维度说明
测试基于四大赛博对立维度：
- **I (Observer)** 潜水 vs **E (Broadcaster)** 显眼
- **P (Zen)** 佛系 vs **S (Striker)** 出击
- **L (Logic)** 严谨 vs **A (Abstract)** 混沌
- **K (Collector)** 仓鼠 vs **W (Surfer)** 冲浪
